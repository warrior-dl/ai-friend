# AI Friend 插件指令列表

## 可用指令

### 1. 订阅管理

#### 开启陪伴
```
/subscribe
/开启陪伴
/订阅
```
订阅每日定时问候推送。

**示例**:
```
用户: /开启陪伴
机器人: 订阅成功！🌟

每天 22:00，我会主动问候你：
"今天感觉怎么样？有什么想分享的吗？"

如果不想收到推送，发送 /关闭陪伴 即可取消~
```

---

#### 关闭陪伴
```
/unsubscribe
/关闭陪伴
/取消订阅
```
取消每日定时问候推送。

**示例**:
```
用户: /关闭陪伴
机器人: 已取消订阅。
如果想再次开启，随时发送 /开启陪伴~
```

---

### 2. 手动触发

#### 夸夸我
```
/praise
/夸夸我
/夸我
```
立即触发夸夸对话，无需等待定时推送。

**示例**:
```
用户: /夸夸我
机器人: 今天做了什么想和我分享的吗？

用户: 就看了两页书
机器人: 太强了！在工作了一整天之后还能坚持看两页书...
```

---

#### 今日总结
```
/summary
/今日总结
/总结
```
今日总结，与 `/夸夸我` 功能相同。

**示例**:
```
用户: /今日总结
机器人: 来说说今天的收获吧，哪怕是很小的一件事~
```

---

## 指令说明

- **英文指令**: 主指令名称（如 `/subscribe`）
- **中文指令**: 别名，更方便记忆和使用（如 `/开启陪伴`）
- **支持前缀**: 所有指令都需要加 `/` 前缀

---

## 常见用法

### 日常使用流程

1. **首次使用**: 发送 `/开启陪伴` 订阅
2. **等待推送**: 每天晚上 22:00 收到问候
3. **随时触发**: 任何时候都可以发送 `/夸夸我`
4. **取消订阅**: 不需要时发送 `/关闭陪伴`

### 快速触发

如果不想等到晚上，可以随时：
```
/夸夸我
```

### 使用英文指令

如果你喜欢简洁的英文指令：
```
/subscribe    # 订阅
/unsubscribe  # 取消订阅
/praise       # 夸夸我
/summary      # 总结
```

## 技术说明

根据 AstrBot 的规范：
- **主指令名**: 使用英文（便于系统识别）
- **别名（alias）**: 使用中文或其他语言（便于用户记忆）
- **装饰器语法**: `@filter.command("主指令", alias={"别名1", "别名2"})`
